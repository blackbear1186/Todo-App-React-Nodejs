Backend - Do in Home folder
1. 'npm init' 
    - Set entry point to server.js 
    - Enter auther and license MIT

2. Install packages: 'npm i express mongoose cors'

3. Install nodemon as a dev dependency: 'npm i -D nodemon'

4. Create server.js

5. Package.json add scripts for starting the server.

// Install react globally
npm install -g create-react-app

// Add Package.json client-install for users to easily install all react dependencies

"npm start --prefix client"
